Pack Camberry


Pack Camberry incluye todo lo que necesitas para poder disfrutar del mejor servicio de atención de vigilancia al mejor precio. Un sistema económico, simple, usable y eficaz.

Pack Camberry incluye una Raspberry Pi con una cámara web incorporada para que puedas monitorizar y vigilar de la forma más sencilla posible. Ambos elementos vienen debídamente configurados para poder funcionar con sólo instalar.


  1. Raspberry Pi

    • Sistema Operativo: Raspbian Wheezy
    • DNS dinámico DuckDNS
    • SSH configurado
    • Iptables configurado


  2. PS3 EyeToy

    • Plug & Play
